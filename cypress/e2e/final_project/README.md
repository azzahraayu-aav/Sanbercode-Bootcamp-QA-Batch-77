# Final Project Cypress Automation Testing - OrangeHRM by Azzahra Ayu

## Deskripsi

Project ini merupakan automation testing menggunakan Cypress pada website OrangeHRM Demo.
Automation dibuat menggunakan beberapa pendekatan:

* Page Object Model (POM)
* Cypress Intercept
* UI Assertion
* API Assertion

Website yang digunakan:

https://opensource-demo.orangehrmlive.com/

---

## Tools & Technology

* Cypress
* JavaScript
* Page Object Model (POM)
* Cypress Intercept
* OrangeHRM Demo

---

## Fitur yang Diotomasi

    1. Login
    2. Directory
    3. Recruitment
