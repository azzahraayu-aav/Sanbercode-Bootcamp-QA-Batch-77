# Tugas 18 - API Automation Testing menggunakan Cypress

## 📌 Deskripsi

Project ini merupakan tugas API Automation Testing menggunakan Cypress pada API Categories dari Platzi Fake API.

Website API:
https://fakeapi.platzi.com/en/rest/categories

API Endpoint:
https://api.escuelajs.co/api/v1/categories

Pengujian dilakukan sebanyak 12 request dengan melakukan assertion pada status code dan response body.

---

## 🛠 Tools

* Cypress
* JavaScript
* Node.js

---

## 🔍 Assertion

Pengujian dilakukan dengan memverifikasi:

### Status Code

* 200 OK

### Response Body

* Response tidak kosong
* Response bertipe Array atau Object
* ID sesuai dengan request
* Slug sesuai dengan request
* Products berhasil ditampilkan

---

## 📊 Hasil Testing

Seluruh test case berhasil dijalankan menggunakan Cypress dengan status:

```bash
12 passing
0 failing
```
